Paste a wiredrive.com URL from the embed form into your post to embed the media.

== Description ==

While viewing a file detail, click "embed" to open the embed form.

Choose your settings and click the button to generate your link.

Copy and paste the share url into your post on its own line or between [embed]yourURLgoesHere[/embed] shortcodes.

The current media embed code is saved and will be displayed once the post is saved.

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory or add via the WordPress Plugins built in plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Paste a Wiredrive.com embed URL into a page or post.
